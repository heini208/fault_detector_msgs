## fault_detector_msgs

The companion package **`fault_detector_msgs`** provides the custom ROS 2 message types used by the **`fault_detector_spot`** stack. It defines:

- `BasicCommand.msg`
- `ComplexCommand.msg`
- `CommandRecordControl.msg`
- `StringArray.msg`
- `TagElement.msg`
- `TagElementArray.msg`

These messages are used for all command, state, and recording interfaces in `fault_detector_spot`.  
Repository: [`fault_detector_spot`](https://github.com/heini208/fault_detector_spot).
